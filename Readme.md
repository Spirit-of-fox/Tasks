
		Руководство пользователя

1. Скачать архив с тестами.
2. Разархивировать его в выбранной папке.
3. Открываем консоль с путём к папке проекта и выполняем пункт 4.
ИЛИ Открываем проект через IDE, к примеру IntelliJ Idea, в терминале выполняем пункт 4.
4. Запускаем командой mvn test.
5. Для запуска в других браузерах необходимо использовать команду mvn test –Dbrowser=firefox, в данном случаи
используется браузер Dbrowser=firefox.
6. Для получения отчёта allure необходимо использовать комманду allure serve allure-results или воспользоватся
плагинами maven.
