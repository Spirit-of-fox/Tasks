
		Руководство пользователя

1. Скачать архив с тестами.
2. Разархивировать его в выбранной папке.
3. Открываем проект через IDE, к примеру IntelliJ Idea в терминале выполняем пункт 4.
* при необходимости производится настройка Program Structure и Build проекта
4. Запускаем командой mvn test.
5. Для получения отчёта allure необходимо использовать комманду allure serve allure-results или воспользоватся плагинами maven.

       Реализуемый сценарий
.
1. Перейти на https://store.steampowered.com/ > Проверить, что язык интерфейса английский, если нет, выбрать его.
2. В меню навести мышь на Categories (Категории) > кликнуть Action (Экшены). Проверить, что количество игр, выводимых на страницу не больше 20 для первой и четвертой страниц.
3. На вкладке New and Trending выбираем игру из условия максимальной скидки (только на первой странице). Если скидки отсутствуют, то выбираем игру с максимальной ценой.
* в случае появления запроса о возрасте (в инкогнито моде на примере https://store.steampowered.com/agecheck/app/34870/), заполняем валидными данными и нажимаем View Page.
4. На странице с описанием игры проверяем корректность цены и скидки (за эталон взять данные из п 3).
5. Кликаем по ссылке Install Steam в правом верхнем углу.
6. Качаем Setup файл (подразумевается ожидание полной загрузки файла).
