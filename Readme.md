Данные тесты предназначены для проверки наличия ключевого слово в блоках результатов поиска и
сравнивание количества результатов поиска сайта с желанным числом пользователя. Тест может
проводиться в двух браузерах, по умолчанию chrome, но так же доступен и firefox.		

		Руководство пользователя

1. Скачать архив с тестами.
2. Разархивировать его в выбранной папке.
3. Открываем консоль с путём к папке проекта и выполняем пункт 4.
ИЛИ Открываем проект через IDE, к примеру IntelliJ Idea, в терминале выполняем пункт 4.
4. Запускаем командой mvn test.
Изменить входные параметры (Слово для поиска, количество результатов для сравнения и браузер) можно
в файле properties.txt расположенном в папке resources, заменив существующие параметры на свои.
